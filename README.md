# 📝 Todo App

A simple and clean Todo App built with React.js during my semester holidays.

## ✨ Features
- ✅ Add new tasks
- ✔️ Mark tasks as done (strikethrough)
- 🗑️ Delete tasks
- 🎨 Clean and minimal UI

## 🛠️ Built With
- React.js
- CSS

## 🚀 Getting Started

Clone the repository:

    git clone https://github.com/sabarnasivanathan-spec/todo-app.git

Install dependencies:

    npm install

Run the app:

    npm start
